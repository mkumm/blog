## Any One Will Do


![Any One Will Do - Wheel](/shorts/wheel.png)

### My randomizer

I built the randomizer I always wished existed — a more “pro” (but still free) way to pick one thing from a larger set of options.

There are plenty of free tools out there, but most are cluttered with ads or feel overly basic.

Any One Will Do is a simple, privacy-first randomizer with a clean UI and smooth, satisfying animations. It includes features like auto-saved sessions, import/export, weighted entries, shared results, grouping, and support for adding metadata (like links) to entries.

No ads. No accounts. No tracking. No upsells.

If you try it, I’d genuinely appreciate any feedback on what could make it better.
